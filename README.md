me-proxy：一个高效的代理管理工具，支持多种协议和动态路由配置，旨在简化网络请求的管理和优化，提升开发效率和系统性能。

#### 安装教程

1. Windows Docker挂载磁盘
```
docker run -v C:\Users\obull\output:/mnt/c/output \
            -v D:\CodeSpace\me-proxy:/mnt/d/CodeSpace/me-proxy \
            your-image-name
```

2. Git 凭据配置
`git push` 需要认证，需要在仓库目录中配置 Git 个人访问令牌：
+ 进入仓库目录，如：`D:\CodeSpace\me-proxy`
+ 执行 `git config credential.helper store`
+ 下次执行 `git push` 时输入一次用户名和 Gitee 个人令牌后，凭据就会被保存。
