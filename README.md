#### 安装教程

1. . Git 凭据配置
`git push` 需要认证，需要在仓库目录中配置 Git 个人访问令牌：
+ 进入仓库目录，如：`D:\CodeSpace\me-proxy`
+ 执行 `git config credential.helper store`
+ 下次执行 `git push` 时输入一次用户名和 Gitee 个人令牌后，凭据就会被保存。
